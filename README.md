# Shadow Fight 4: Arena — Tournament Bot

Telegram-бот для организации боёв, турниров и челленджей в игре Shadow Fight 4: Arena.

## Возможности

### Бои
- **Создание боя** — по Game ID из игры, с выбором типа (дружеский/рейтинговый) и персонажа
- **Присоединение** — поиск открытых боёв других игроков, указание своего персонажа
- **Завершение** — выбор победителя, скриншоты результата
- **Статистика** — винрейт, рейтинг, серии побед, статистика по персонажам

### Турниры
- Single Elimination (олимпийская система)
- Автоматическая генерация сетки с bye'ами
- `/tournament_new` — создание турнира (админ)
- `/tournament_start` — старт с рассылкой уведомлений
- `/tadvance` — проведение раунда
- `/bracket` — просмотр сетки

### Достижения (8 штук)
- Первая победа, Ветеран (10 побед), Боец (50 боёв), Легенда (100 боёв)
- На грани (5 подряд), Неудержимый (10 подряд)
- Чемпион (выиграть турнир), Мастер персонажа (10 побед одним)

### Голосования
- `/poll` — создание опроса с вариантами (админ)
- Интерактивные кнопки с live-прогрессом
- Закрытие админом с подведением итогов

### Профиль и рейтинг
- `/profile` — статистика, персонажи, достижения
- `/leaderboard` — топ-20 по рейтингу
- `/achievements` — список достижений

### Администрирование
- `/announce` — объявление в чат
- `/challenge` — создание челленджа
- `/ban /unban` — бан игроков
- `/backup` — скачать базу данных
- `/restore` — восстановить базу из файла

## Быстрый старт

```bash
git clone https://github.com/aibigwinner/tsfa.git
cd tsfa
pip install -r requirements.txt
```

Создать `.env`:
```
BOT_TOKEN=1234567890:ABCdefGHIjklmNOPqrstUVwxyz
ADMIN_IDS=7153815329
```

Запуск локально:
```bash
python main.py
```

## Деплой на Render

1. Создать Web Service на Render, подключить GitHub-репозиторий
2. Указать:
   - **Build Command:** `pip install -r requirements.txt`
   - **Start Command:** `python app.py`
3. Добавить Environment Variables:
   - `BOT_TOKEN`
   - `ADMIN_IDS`
   - `RENDER_EXTERNAL_URL` = `https://shadow-fight-arena-bot.onrender.com`

### Предотвращение «засыпания» (cron)

Render free web services **засыпают** через 15 минут бездействия. Чтобы бот был всегда онлайн:

1. Зарегистрироваться на [cron-job.org](https://cron-job.org) (бесплатно)
2. Создать задачу:
   - **URL:** `https://shadow-fight-arena-bot.onrender.com/`
   - **Interval:** Every 10 minutes
   - **Method:** GET
3. Сохранить

Любой HTTP-запрос к сервису (даже с ответом 404) сбрасывает таймер бездействия Render.

## Резервное копирование

Render пересоздаёт файловую систему при каждом деплое — данные теряются.

**Перед деплоем** → в чате с ботом:
```
/backup
```
Бот пришлёт файл `tsfa_backup.db` — сохрани его.

**После деплоя** → отправь файл боту:
```
/restore
```
→ прикрепи файл `tsfa_backup.db`

## Хранение данных

- SQLite (`data/bot.db`) — ACID-транзакции, thread-safe
- Автомиграция из JSON при первом запуске
- `/backup` и `/restore` для переноса между деплоями
