## 🚀 Быстрый старт


## 👤 Команды для всех

### `/start`
Приветственное сообщение со списком основных команд.

### `/help`
Полный список всех доступных команд (для админов — расширенный).

### `/profile`
**Файл:** `handlers/profile.py`

Показывает профиль игрока:
- Имя, username
- Рейтинг ⭐
- Победы / Поражения / Винрейт (WR)
- Всего боёв
- Текущая серия побед 🔥
- Количество достижений 🎖️
- Роль (Админ / Игрок)
- Статистика по персонажам
- Количество активных боёв

Кнопки: «📊 Топ игроков», «🎖️ Достижения»

### `/leaderboard`
**Файл:** `handlers/profile.py:53`

Топ-20 игроков по рейтингу. Показывает место, имя, рейтинг, победы/поражения и винрейт.

### `/battle`
**Файл:** `handlers/battle.py`

Главное меню управления боями. Кнопки:
- **🎮 Создать бой** — ввод Game ID → выбор типа (дружеский/рейтинговый) → выбор персонажа
- **🔍 Найти бой** — список открытых боёв с Game ID и персонажем создателя
- **📸 Загрузить скриншот** — загрузить доказательство результата
- **✅ Завершить бой** — выбрать активный бой → указать победителя
- **❌ Отменить бой** — отменить свой бой (только создатель)

### `/battles`
Список всех активных боёв в чате.

### `/tournament`
**Файл:** `handlers/tournament.py`

Список турниров. Для каждого показывает:
- Статус (📝 регистрация / ⚔️ идёт / 🏆 завершён)
- Количество зарегистрированных / максимум
- Кнопка «📝 Записаться» если идёт регистрация
- Кнопка «📋 Сетка» если турнир начался

### `/bracket <id турнира>`
Показать сетку конкретного турнира в текстовом формате.

### `/achievements`
**Файл:** `handlers/achievements.py`

Список всех достижений с отметкой о получении:

| Иконка | Название | Условие |
|--------|----------|---------|
| 🏆 | Первая победа | 1 победа |
| 🎖️ | Ветеран | 10 побед |
| ⚔️ | Боец | 50 боёв |
| 🔥 | На грани | 5 побед подряд |
| 💥 | Неудержимый | 10 побед подряд |
| 👑 | Чемпион | Выиграть турнир |
| 🎯 | Мастер персонажа | 10 побед одним персонажем |
| 💎 | Легенда | 100 боёв |

**Как добавить своё достижение:**
Открой `handlers/achievements.py`, в список `ACHIEVEMENTS` добавь:
```python
{
    "id": "unique_id",
    "name": "Название",
    "desc": "Описание условия",
    "icon": "🏅",
    "check": lambda p: p.get("wins", 0) >= 25,
}
```

## 🔧 Команды для администраторов

Все админские права получают пользователи, чьи ID указаны в переменной `ADMIN_IDS` в `.env`.

### `/announce <текст>`
**Файл:** `handlers/admin.py:7`

Сделать объявление в чат. Текст отправляется от имени бота, исходное сообщение удаляется.

### `/challenge <название> | <описание> | <цель>`
**Файл:** `handlers/admin.py:36`

Создать челлендж/ивент. Пример:
```
/challenge 10 побед Рэйденом | Одержи 10 побед за Рэйдена | 10
```

### `/tournament_new <название> [макс. игроков]`
**Файл:** `handlers/admin.py:78`

Создать новый турнир. По умолчанию 8 игроков, можно указать любое число.

Пример:
```
/tournament_new Кубок Сезона 8    # на 8 игроков
/tournament_new Мини-турнир 4      # на 4 игроков
```

### `/tournament_start <id турнира>`
**Файл:** `handlers/tournament.py:61`

Начать турнир: закрыть регистрацию, перемешать игроков, сгенерировать сетку.
Если игроков не степень двойки — топ-игроки получают bye (авто-проход в следующий раунд).
Всем участникам приходит уведомление.

### `/tadvance <id турнира> <номер матча> <id победителя>`
**Файл:** `handlers/tournament.py:79`

Провести матч в турнире. Номер матча начинается с 0.

Пример для первого матча:
```
/tadvance tournament_1712345678 0 987654321
```

Когда все матчи раунда сыграны — автоматически создаётся следующий раунд.
Когда остаётся один матч и он сыгран — турнир завершается, победитель получает достижение «👑 Чемпион».

### `/ban <user_id>` / `/unban <user_id>`
**Файл:** `handlers/admin.py:121`

Забанить / разбанить игрока по User ID.

### `/poll <вопрос> | <вариант1> | <вариант2> | ...`
**Файл:** `handlers/voting.py`

Создать голосование. От 2 до 9 вариантов.

Пример:
```
/poll Какой следующий челлендж? | 10 побед Рэйденом | 20 побед Лин | 5 побед любым
```

После создания появляется голосование с кнопками. При голосовании счётчик обновляется.
Кнопка «🔒 Закрыть» завершает голосование и показывает результаты.

### `/poll_close <id голосования>`
Закрыть голосование вручную (если кнопка не сработала).

## ⚔️ Система боёв

### Жизненный цикл боя

```
1. Создание:  /battle → «Создать бой» → Game ID → тип → персонаж
2. Поиск:     /battle → «Найти бой» → выбрать → указать персонажа
3. Бой:       статус "active", Game ID доступен обоим
4. Скриншот:  /battle → «Загрузить скриншот» → выбрать бой → отправить фото
5. Завершение: /battle → «Завершить бой» → выбрать → указать победителя
6. Отмена:    /battle → «Отменить бой» → выбрать (только создатель)
```

### Формат Game ID
Game ID — это идентификатор боя из игры Shadow Fight 4: Arena. Игрок создаёт бой в игре и копирует ID оттуда.

### Завершение боя
После завершения автоматически:
- Обновляется счёт побед/поражений 📊
- Обновляется рейтинг ⭐
- Обновляется серия побед 🔥
- Проверяются и выдаются достижения 🎖️
- Обновляется статистика персонажа 🎯
- Победителю приходит уведомление о новом достижении

## 🏆 Турниры

### Формат
Single Elimination (олимпийская система). 
- Если игроков не степень двойки — случайные игроки получают bye (авто-проход).
- Админ вручную проводит каждый матч через `/tadvance`.
- Когда все матчи раунда сыграны — следующий раунд создаётся автоматически.
- Финальный матч определяет чемпиона турнира.

### Пошаговый сценарий

```
1. Админ:    /tournament_new Кубок Сезона 8
   → Бот:   Турнир создан, ID: tournament_1712345678

2. Игроки:  /tournament → «📝 Записаться: Кубок Сезона»

3. Админ:    /tournament_start tournament_1712345678
   → Бот:   Сетка сгенерирована, всем участникам уведомление

4. Админ:    /tadvance tournament_1712345678 0 111111111
   → Бот:   Победитель матча #1 записан

5. Админ:    /tadvance tournament_1712345678 1 222222222
   → Бот:   Раунд 1 завершён! Начинается раунд 2.

6. ...повторять пока не останется один победитель

7. Бот:     🏆 Турнир завершён! Победитель: Имя
            Победитель получает достижение «👑 Чемпион»
```

## 🎖️ Достижения

### Как это работает
**Файл:** `handlers/achievements.py`

После каждого завершённого боя вызывается `check_and_award(user_id)`, которая проверяет все достижения. Если условие выполнено и достижение ещё не было получено — оно выдаётся, и игроку приходит уведомление.

### Добавление новых достижений
В `handlers/achievements.py` в список `ACHIEVEMENTS`:
```python
ACHIEVEMENTS = [
    # ... существующие ...
    {
        "id": "twenty_five_wins",          # уникальный ID
        "name": "Бессмертный",             # название
        "desc": "25 побед",                # описание
        "icon": "🏅",                      # иконка
        "check": lambda p: p.get("wins", 0) >= 25,  # функция-проверка
    },
]
```

`p` — это словарь игрока из `players.json`.

## 📊 Голосования

**Файл:** `handlers/voting.py`

Создаются админом для принятия коллективных решений (например, какой следующий челлендж).

### Особенности
- Голоса хранятся в `data/polls.json`
- При голосовании прогресс-бар обновляется в реальном времени
- Результаты: количество голосов + процент
- Закрыть может только админ

## 🌐 Деплой на Render

### Через render.yaml (рекомендуется)

1. Создай репозиторий на GitHub
2. Заливай проект:
```bash
git init
git add .
git commit -m "Первый коммит"
git remote add origin <url>
git push -u origin main
```

3. Зайди на [Render Dashboard](https://dashboard.render.com)
4. New → Web Service → Connect your GitHub repo
5. Render сам прочитает `render.yaml`
6. Добавь переменные окружения:
   - `BOT_TOKEN` — токен от @BotFather
   - `ADMIN_IDS` — ID администраторов через запятую (узнать: @userinfobot)
7. Выбери план **Free**
8. Deploy 🚀

### Вручную

Если `render.yaml` не подхватился:
- **Runtime:** Python
- **Build Command:** `pip install -r requirements.txt`
- **Start Command:** `python app.py`
- **Plan:** Free

### Как получить BOT_TOKEN
1. Напиши [@BotFather](https://t.me/BotFather) в Telegram
2. `/newbot` → название → username
3. Скопируй токен вида `1234567890:ABCdef...`

### Как узнать свой User ID
1. Напиши [@userinfobot](https://t.me/userinfobot)
2. Он покажет твой числовой ID

## 🐛 Решение проблем

**Бот не отвечает на команды:**
- Проверь `BOT_TOKEN` в .env
- Убедись, что бот добавлен в группу/чат
- Проверь логи: `python main.py`

**Не работают админ-команды:**
- Проверь `ADMIN_IDS` в .env
- Убедись, что твой ID совпадает (узнать через @userinfobot)

**Ошибка при импорте:**
```bash
pip install -r requirements.txt
```

**На Render бот не стартует:**
- Проверь логи в Dashboard → Logs
- Убедись, что `BOT_TOKEN` и `ADMIN_IDS` добавлены в Environment Variables
- Убедись, что Start Command: `python app.py`
